Приложение позволяет получать ежедневные обновления о текущих ценах на электроэнергию на рынке ЧР. Вы можете добавлять к этим ценам распределительные затраты за каждый час и определять часы, в которые вы используете низкий тариф. На основе текущей цены или ценового индекса вы можете управлять своими потоками и отслеживать цены с помощью интерактивных виджетов.

Для правильной работы:
1. Добавьте устройство в Homey
2. Установите распределительные затраты для высокого и низкого тарифов
3. Укажите часы, когда доступен низкий тариф
4. При необходимости включите ведение логов для отладки

Функции приложения:
- Мониторинг цен в реальном времени через два виджета:
  - Виджет Текущей Часовой Цены, показывающий текущие, цены на следующий час и средние цены
  - Интерактивный График Дневных Цен с цветовой кодировкой тарифов и уровней цен
- Автоматическое переключение на резервный источник данных, если основной API недоступен
- Отображение цен в МВтч или кВтч
- Полная поддержка потоков для автоматизации на основе цен и тарифов
- Продвинутое вычисление оптимальных периодов цен

Распределительные затраты будут автоматически включены в цены при следующем обновлении. Приложение использует два источника данных: основные данные с spotovaelektrina.cz и резервные данные от OTE-CR при необходимости.
