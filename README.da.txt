Applikationen giver dig mulighed for at modtage daglige opdateringer om de aktuelle spot elpriser for CZ-markedet. Den giver dig mulighed for at tilføje distributionsomkostninger til disse priser for hver time og definere de timer, hvor du benytter den lavere tarif. Baseret på den aktuelle pris eller prisindeks kan du kontrollere dine strømme og overvåge priser gennem interaktive widgets.

For korrekt funktionalitet:
1. Tilføj enheden til Homey
2. Indstil distributionsomkostningerne for både høje og lave tariffer
3. Angiv de timer, hvor den lavere tarif er tilgængelig
4. Du kan valgfrit aktivere fejlfindingslogning til fejlsøgning

Appens funktioner:
- Realtidsprismonitorering gennem to widgets:
  - Aktuel Timepris Widget, der viser aktuelle, næste time og gennemsnitspriser
  - Interaktiv Daglig Prisgraf med farvekodede tariffer og prisniveauer
- Automatisk skift til sikkerhedsdatakilde, hvis det primære API er utilgængeligt
- Prisvisning i enten MWh eller kWh
- Omfattende flowstøtte til automatisering baseret på priser og tariffer
- Avanceret beregning af optimale prisperioder

Distributionsomkostningerne vil automatisk blive inkluderet i priserne ved næste opdatering. Appen bruger to datakilder: primær data fra spotovaelektrina.cz og backup data fra OTE-CR når nødvendigt.
